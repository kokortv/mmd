# Unda

Unda — минималистичное приложение для списка покупок. Оно помогает быстро собрать список, поделиться им с близкими, отметить купленное в магазине и не тратить лишнее время на ручное редактирование.

Приложение работает как PWA: его можно открыть в браузере или установить на телефон как обычное приложение.

## Что Умеет Unda

- Быстро добавлять товары через одно поле ввода.
- Подсказывать продукты из общей базы.
- Указывать количество сразу при вводе.
- Отмечать купленные товары нажатием на карточку или чекбокс.
- Автоматически переносить купленное вниз списка.
- Удалять товары свайпом влево или кнопкой `x`.
- Возвращать случайно удаленное через `Вернуть`.
- Очищать список только после подтверждения.
- Делиться списком по ссылке.
- Синхронизировать список между устройствами.
- Работать устойчивее при плохом интернете: изменения сначала сохраняются локально, а потом отправляются на синхронизацию.

## Быстрое Добавление

Начни вводить название товара, и Unda покажет подсказки. Если нужный товар появился в подсказке, его можно выбрать.

На компьютере подсказками можно управлять с клавиатуры:

```text
ArrowDown / ArrowUp — выбрать подсказку
Enter — добавить товар
```

Если товара еще нет в базе, просто введи его название и нажми `Enter` или `+`.

## Умный Ввод Количества

Количество можно писать прямо вместе с названием товара.

Например:

```text
молоко2л
яйца10шт
кефир 2,5%1шт
масло сливочное 2 пачки
```

Unda сама отделит название от количества:

```text
Молоко              2 л
Яйца                10 шт
Кефир 2,5%          1 шт
Масло сливочное     2 пачки
```

Приложение также исправляет частые варианты написания:

```text
2л    -> 2 л
2шт   -> 2 шт
2, 5% -> 2,5%
2. 5% -> 2.5%
```

## Привычные Количества

Unda запоминает часто используемые количества. Если ты часто добавляешь `Молоко 2 л`, то в следующий раз при вводе `мол...` подсказка может сразу показать `2 л`.

Это помогает быстрее собирать повторяющиеся списки.

## Важные И Сомнительные Товары

Для особых товаров можно использовать символы `!` и `?`.

```text
хлеб!
сыр?
```

Символы не попадают в название товара.

Товар с `!` считается важным:

- держится сверху списка, пока не куплен;
- выглядит крупнее обычного;
- не перетаскивается вручную;
- помечается маленьким `!`.

Товар с `?` помечается как товар под вопросом. Это удобно для покупок в стиле “если будет нормальная цена” или “если вспомню, зачем он нужен”.

## Количество В Карточке

Количество можно добавить или изменить не только при вводе, но и через карточку товара.

Нажми маленькую кнопку количества на активной карточке и введи, например:

```text
2 л
1 кг
300 г
2 пачки
```

Если товар уже отмечен купленным, количество больше не редактируется. Такой товар можно только вернуть в активные или удалить.

## Купленные Товары

Товар можно отметить купленным нажатием на карточку или чекбокс.

После этого он:

- плавно перемещается вниз;
- становится визуально спокойнее;
- перестает участвовать в ручной сортировке;
- остается в списке некоторое время, чтобы его можно было проверить.

Позже купленные товары автоматически исчезают из текущего списка, но остаются в базе подсказок.

## Сортировка И Перетаскивание

Есть два режима порядка:

- по алфавиту;
- по добавлению / вручную.

Если выбран порядок по добавлению, товары можно перетаскивать вверх и вниз.

Важные и купленные товары вручную не перетаскиваются: важные держатся сверху, купленные — внизу.

## Удаление И Очистка

Товар можно удалить свайпом влево или кнопкой `x`.

После удаления появляется короткая возможность вернуть товар через `Вернуть`.

Для очистки всего списка есть отдельная кнопка `⌫`. Перед очисткой Unda показывает подтверждение, чтобы случайный тап не стер список.

## Общий Список

Списком можно поделиться по ссылке через кнопку `🔗`.

Все, кто открывает одну и ту же ссылку, видят один общий список. Можно вместе добавлять товары, удалять лишнее и отмечать купленное.

Это удобно, когда один человек составляет список дома, а другой идет в магазин.

## Синхронизация

Кнопка `⇅` синхронизирует список.

Если интернет нестабильный, Unda сначала сохраняет изменения локально. Когда соединение восстановится, изменения отправятся на синхронизацию.

Статус синхронизации показывается маленьким индикатором рядом с названием приложения и цветом кнопки `⇅`.

## Установка На Телефон

Unda можно использовать как обычное PWA-приложение: открыть в браузере и добавить на главный экран.

После установки приложение открывается в отдельном окне и ощущается ближе к обычному мобильному приложению.
