# Unda

Минималистичное PWA-приложение для списка покупок. Интерфейс работает сразу в демо-режиме, а после настройки Google Apps Script читает и пишет данные в Google Sheets.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открой `http://localhost:4173`.

## Подключение Google Sheets

1. Создай новую Google Таблицу.
2. Открой `Extensions -> Apps Script`.
3. Вставь код из `apps-script/Code.gs`.
4. Нажми `Deploy -> New deployment -> Web app`.
5. Для `Execute as` выбери себя, для `Who has access` выбери нужный режим доступа.
6. Скопируй Web app URL.
7. Вставь URL в `config.js`:

```js
window.SHOPPING_APP_CONFIG = {
  appsScriptUrl: "https://script.google.com/macros/s/AKfycbxpc9x7xryWbRmS2pLT4h_kcIQOxFkW4Xu2LoDc2rLu5jI7cw3Axo3TVpPmjIAHJKTBEA/exec",
  shareBaseUrl: ""
};
```

`shareBaseUrl` можно оставить пустым: приложение само возьмет текущий публичный адрес страницы. Заполнять его нужно только если хочешь принудительно шарить другой домен.

Apps Script сам создаст листы `Products` и `Items`, если их еще нет. Разные ссылки не создают новые вкладки в таблице: все списки лежат в одном листе `Items`, а разделяются колонкой `listId`.

## Структура таблицы

`Products`

| name |
| --- |
| Молоко |

`Items`

| id | name | bought | createdAt | sortOrder | boughtAt | quantity | listId | marker |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |

## Что уже есть

- Подсказки из базы товаров.
- Добавление новых товаров в список и базу.
- Количество у товара: можно сразу ввести `масло сливочное 2 пачки` или нажать на маленькую кнопку на карточке и указать `2`, `1 кг`, `300 г`.
- Важные товары: `молоко!` выделяется, держится наверху и не перетаскивается, `сыр?` помечается как товар под вопросом. Символы в название не попадают.
- Сортировка по добавлению или по алфавиту.
- Ручное перетаскивание товаров в режиме сортировки по добавлению.
- Выбор подсказок с клавиатуры: стрелки вверх/вниз и Enter.
- Отметка купленного через чекбокс.
- Купленные товары автоматически удаляются из списка через 3 дня, но остаются в базе подсказок.
- Очистка текущего списка кнопкой с губкой.
- Удаление кнопкой или свайпом влево.
- Шаринг списка по ссылке `?list=...`; все с одной ссылкой редактируют один список, разные ссылки не мешают друг другу.
- PWA manifest и сервис-воркер для установки на устройство.
